# Dream Hack Proje İskeleti

Türkiye pazarına uygun, para kazandıracak modern ve ölçeklenebilir bir web uygulaması altyapısı.

## Ana Modüller
- Kullanıcı Yönetimi
- Ödeme Entegrasyonu (iyzico/PayTR)
- Yönetim Paneli
- Çoklu Dil Desteği (TR/EN)
- Frontend: Next.js (React)
- Backend: Express.js (Node.js)
- Veritabanı: PostgreSQL

## Klasör Yapısı
- apps/frontend: Next.js tabanlı frontend
- apps/backend: Express.js tabanlı backend
- packages/common: Ortak kod ve tipler
- database: Veritabanı şemaları ve migration dosyaları
